# 📈 设备保养数据分析器

> pandas + matplotlib 练手项目 — 用真实场景学数据分析

## 练了什么

| pandas 技能 | 在哪体现 |
|-------------|---------|
| 读 CSV | `pd.read_csv()` 加载保养记录 |
| 查看数据 | `df.head()` / `df.info()` / `df.describe()` |
| 数据清洗 | 处理空值、转日期格式、去重 |
| 分组聚合 | `df.groupby().agg()` 按产线/类型统计 |
| 排序 | `df.sort_values()` 按保养次数排序 |
| 时间处理 | `pd.to_datetime()` + `.dt.month` 提取月份 |
| 列表推导 | 按位置/类型/状态筛选设备 |
| 写 CSV | `df.to_csv()` 导出分析结果 |

| matplotlib 技能 | 在哪体现 |
|----------------|---------|
| 柱状图 | 各产线保养次数对比 |
| 折线图（双Y轴） | 月度保养趋势 |
| 饼图 | 保养类型分布占比 |
| 中文配置 | SimHei 字体设置 |

## 运行方式

```bash
pip install pandas matplotlib
python main.py
```

首次运行会自动生成模拟数据（`maintenance_data.csv`），有真实数据替换即可。
